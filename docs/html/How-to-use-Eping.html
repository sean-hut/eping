3 How to use Eping

Eping is used through it’s eping command.

3.1 Minibuffer Output

M-x eping

You will be promted for a domain to ping and then for a number of times to ping the domain. Eping then pings the selected domain the specified number of times with ping. Eping outputs to the minibuffer the success or failure of ping to ping the domain.

3.2 eSpeak Output

C-u M-x eping

You will be promted for a domain to ping and then for a number of times to ping the domain. Eping then pings the selected domain the specified number of times with ping. Eping will say the success or failure of ping to ping the domain using espeak.
